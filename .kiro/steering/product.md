# Product Overview

A responsive mobile app landing page showcasing the "Luxi Mobile" application. The landing page serves as a marketing website to promote app downloads and features.

## Key Features

- Hero section with app preview and download CTAs
- Statistics counter (downloads, transactions, ratings)
- Feature showcase with alternating layouts
- App screenshot gallery
- Customer testimonials with profile carousel
- FAQ accordion section
- Partner/client logo grid
- Contact form modal
- Multi-language support
- Dark mode toggle with persistent preference

## Target Audience

Potential mobile app users looking to learn about and download the Luxi Mobile application from App Store or Google Play.

## Brand Identity

- Primary brand color: Purple (#673ab7)
- Secondary accent: Cyan
- Modern, clean aesthetic with gradient backgrounds
- Professional yet approachable tone
- Supports light and dark themes for user preference
